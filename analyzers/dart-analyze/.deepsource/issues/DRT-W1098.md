**DON'T** use adjacent strings in a list.

This can indicate a forgotten comma.

**BAD:**
```dart
List<String> list = <String>[
  'a'
  'b',
  'c',
];
```

**GOOD:**
```dart
List<String> list = <String>[
  'a' +
  'b',
  'c',
];
```

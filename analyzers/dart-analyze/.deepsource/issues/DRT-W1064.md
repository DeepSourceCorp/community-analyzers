---
title: "Unnecessary string interpolation."
verbose_name: "unnecessary_string_interpolations"
category: "antipattern"
weight: 70
severity: "major"
---
**DON'T** use string interpolation if there's only a string expression in it.

**BAD:**
```dart
String message;
String o = '$message';
```

**GOOD:**
```dart
String message;
String o = message;
```

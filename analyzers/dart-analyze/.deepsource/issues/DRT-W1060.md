---
title: "Use `=` to separate a named parameter from its default value."
verbose_name: "prefer_equal_for_default_values"
category: "antipattern"
weight: 70
severity: "major"
---
**DO** use `=` to separate a named parameter from its default value.

**BAD:**
```dart
m({a: 1})
```

**GOOD:**
```dart
m({a = 1})
```

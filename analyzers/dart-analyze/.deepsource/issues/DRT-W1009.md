---
title: "Put required named parameters first."
verbose_name: "always_put_required_named_parameters_first"
category: "antipattern"
weight: 70
severity: "major"
---
**DO** specify `required` on named parameter before other named parameters.

**BAD:**
```dart
m({b, c, required a}) ;
```

**GOOD:**
```dart
m({required a, b, c}) ;
```

**BAD:**
```dart
m({b, c, @required a}) ;
```

**GOOD:**
```dart
m({@required a, b, c}) ;
```

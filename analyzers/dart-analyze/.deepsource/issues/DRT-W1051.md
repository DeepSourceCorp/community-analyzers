---
title: "Prefer putting asserts in initializer lists."
verbose_name: "prefer_asserts_in_initializer_lists"
category: "antipattern"
weight: 70
severity: "major"
---
**DO** put asserts in initializer lists.

**BAD:**
```dart
class A {
  A(int a) {
    assert(a != 0);
  }
}
```

**GOOD:**
```dart
class A {
  A(int a) : assert(a != 0);
}
```

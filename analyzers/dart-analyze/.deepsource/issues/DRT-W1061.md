title: "Don't create a lambda when a tear-off will do."
verbose_name: "unnecessary_lambdas"

**DON'T** create a lambda when a tear-off will do.

**BAD:**
```dart
names.forEach((name) {
  print(name);
});
```

**GOOD:**
```dart
names.forEach(print);
```

---
title: "Unnecessary null checks."
verbose_name: "unnecessary_null_checks"
category: "antipattern"
weight: 70
severity: "major"
---
**DON'T** apply a null check when a nullable value is accepted.

**BAD:**
```dart
f(int? i) {}
m() {
  int? j;
  f(j!);
}

```

**GOOD:**
```dart
f(int? i) {}
m() {
  int? j;
  f(j);
}
```

---
title: "Avoid using `null` in `if null` operators."

**AVOID** using `null` as an operand in `if null` operators.

Using `null` in an `if null` operator is redundant, regardless of which side
`null` is used on.

**BAD:**
```dart
var x = a ?? null;
var y = null ?? 1;
```

**GOOD:**
```dart
var x = a ?? 1;
```

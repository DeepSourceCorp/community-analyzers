---
title: "Prefer using if null operators"
verbose_name: "prefer_if_null_operators"
category: "antipattern"
weight: 70
severity: "major"
---
**PREFER** using if null operators instead of null checks in conditional
expressions.

**BAD:**
```dart
v = a == null ? b : a;
```

**GOOD:**
```dart
v = a ?? b;
```

---
title: "Unnecessary new keyword"
verbose_name: "unnecessary_new"
category: "antipattern"
weight: 70
severity: "major"
---
**AVOID** new keyword to create instances.

**BAD:**
```dart
class A { A(); }
m(){
  final a = new A();
}
```

**GOOD:**
```dart
class A { A(); }
m(){
  final a = A();
}
```
